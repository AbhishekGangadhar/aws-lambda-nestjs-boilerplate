# AWS Lambda NestJs Boilerplate

Boilerplate code for using NestJs with AWS Lambda

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Create Lambda Zip

```bash
$ bash create_lambda_zip.sh
```

This command will create a zip file called "nest-lambda.zip"
